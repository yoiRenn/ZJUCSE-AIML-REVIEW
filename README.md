## 核心功能 (Features)

* **AI 语义阅卷**：基于 Qwen-14B 模型（可自选），支持同义词识别、漏答补全、错词划除。
* **智能复习算法**：优先复习错题与模糊题，科学安排复习队列。
* **错题花名册**：按错误频次倒序排列，支持高频错题（3次+）高亮与针对性突击。
* **优化体验**：MathJax 公式渲染、训练数据导入导出（JSON）、纯键盘操作。

##  使用方法 (Usage)

1.  **获取 Key**：前往 [SiliconFlow](https://siliconflow.cn/) 申请免费 API Key (`sk-xxxx`)。
2.  **配置**：点击网页右上角 **⚙️ 设置**，填入 Key 并保存。
3.  **开始**：左上角选择章节 -> **🚀 开始死磕**。

##  部署指南 (Deployment)

如果要部署到 GitHub Pages，**请务必注意以下两点**，否则会报错：

1.  **大小写敏感**：GitHub 服务器是 Linux 环境。
    * 文件夹必须叫 **`data`** (全小写)。
    * 配置文件必须叫 **`config.json`** (全小写)。
2.  **文件过滤**：
    * 在仓库根目录新建一个名为 **`.nojekyll`** 的空文件（防止 GitHub 忽略下划线文件或特定目录）。
